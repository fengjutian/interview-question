## React Native 的核心组件有哪些？
- React Native 的一些核心组件包括：
- View：一个支持布局和样式的容器。
- Text：用于显示文本。
- Image：用于显示图片。
- ScrollView：一个可以容纳多个组件和视图的滚动容器。
- FlatList：一个高效渲染大量数据列表的接口。
