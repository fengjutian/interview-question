## **一、基础题（HTML5 & API）**

1. **HTML5 `<audio>` 与 `<video>` 标签的基本属性有哪些？**

   * **常见属性**：`src`、`controls`、`autoplay`、`loop`、`muted`、`preload`
   * **追问**：

     * `preload` 的不同值（`auto` / `metadata` / `none`）有什么区别？
     * 如何控制播放进度和音量？

2. **前端如何获取视频的播放状态或时间？**

   * 使用事件和属性：

     ```js
     video.currentTime; // 当前播放时间
     video.duration;    // 总时长
     video.paused;      // 是否暂停
     video.ended;       // 是否播放结束
     ```
   * 事件：`play`、`pause`、`ended`、`timeupdate`、`loadedmetadata`

3. **`<video>` 与 Flash 视频有什么区别？**

   * HTML5 原生支持、无需插件
   * 可通过 JS 直接控制
   * 更安全，性能更好，兼容移动端
   * **追问**：为什么移动端不支持 Flash？

4. **前端如何播放流媒体（直播）？**

   * HLS（`.m3u8`）可以用 **video.js + hls.js** 在浏览器播放
   * RTMP 一般需要通过 **Media Server + WebRTC** 转码
   * **追问**：HLS 和 DASH 有什么区别？

## **二、进阶题（浏览器 API & 兼容性）**

5. **MediaSource API 是什么？有什么应用场景？**

   * 允许 JS 动态生成视频流并喂给 `<video>` 播放
   * 应用：点播自适应码率（ABR）、视频剪辑、P2P 视频播放

6. **Web Audio API 可以做什么？**

   * 音频可视化（频谱、波形）
   * 音效处理（增益、滤波器、3D 音效）
   * **追问**：如何实现实时音频可视化？

7. **前端如何获取麦克风或摄像头数据？**

   ```js
   navigator.mediaDevices.getUserMedia({ audio: true, video: true })
     .then(stream => { video.srcObject = stream; });
   ```

   * **追问**：

     * 如何处理权限被拒绝？
     * 如何实现美颜或滤镜效果？

8. **如何实现视频截图？**

   ```js
   const canvas = document.createElement('canvas');
   canvas.width = video.videoWidth;
   canvas.height = video.videoHeight;
   canvas.getContext('2d').drawImage(video, 0, 0);
   const imgData = canvas.toDataURL('image/png');
   ```

9. **跨域视频播放如何解决？**

   * 设置 `video.crossOrigin = "anonymous"`
   * 服务器设置 `Access-Control-Allow-Origin`
   * **追问**：如果是 HLS 或 DASH 流，如何处理？

## **三、实战题（性能优化 & 项目经验）**

10. **如何优化页面视频加载性能？**

    * 使用懒加载（IntersectionObserver）
    * 分段加载 / MediaSource
    * 压缩视频（编码优化）
    * **追问**：为什么 autoplay 视频要 muted 才能在移动端自动播放？

11. **如何实现视频播放进度的断点续播？**

    * 使用 `localStorage` 或 IndexedDB 保存 `currentTime`
    * 视频重新加载时设置 `video.currentTime`

12. **直播延迟太高如何优化？**

    * 选择低延迟编码（H.264 + WebRTC）
    * 减少分段时长（HLS/DASH）
    * 优化网络传输（CDN、WebRTC P2P）

13. **前端如何实现视频拖拽快进和缓冲优化？**

    * 监听 `seeked` 和 `seeking` 事件
    * 结合 `video.buffered` 判断可用缓冲区
    * **追问**：浏览器对大文件 seek 的机制？

14. **移动端自动播放策略限制？**

    * Chrome/iOS Safari 都要求静音或用户交互
    * **追问**：为什么不同浏览器策略不一致？

## **四、拓展题（高阶）**

15. **WebRTC 传输原理和前端实现思路？**

    * P2P 直接传输
    * 信令服务器协助建立连接
    * `RTCPeerConnection`、`getUserMedia`、`RTCDataChannel`
    * **追问**：如何处理网络穿透（NAT/防火墙）？

16. **视频播放卡顿如何排查？**

    * 查看 `buffered` / `readyState`
    * 网络带宽限制 / 分段下载
    * 浏览器渲染瓶颈（Canvas/WebGL 绘制）

17. **音视频同步问题如何解决？**

    * `AudioContext` + `video.currentTime` 校准
    * 直播低延迟：时间戳同步
    * **追问**：延迟不同步会有哪些视觉或听觉问题？
